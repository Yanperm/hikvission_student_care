# การมีส่วนร่วมในโปรเจค

ขอบคุณที่สนใจมีส่วนร่วมในโปรเจค Student Care!

## วิธีการมีส่วนร่วม

1. Fork โปรเจคนี้
2. สร้าง Feature Branch (`git checkout -b feature/AmazingFeature`)
3. Commit การเปลี่ยนแปลง (`git commit -m 'Add some AmazingFeature'`)
4. Push ไปยัง Branch (`git push origin feature/AmazingFeature`)
5. เปิด Pull Request

## แนวทางการเขียนโค้ด

- ใช้ Python PEP 8 style guide
- เขียน docstrings สำหรับ functions และ classes
- เพิ่ม comments สำหรับโค้ดที่ซับซ้อน
- ทดสอบโค้ดก่อน commit

## การรายงานปัญหา

หากพบ bug หรือมีข้อเสนอแนะ กรุณาเปิด Issue พร้อมรายละเอียด:
- คำอธิบายปัญหา
- ขั้นตอนการทำซ้ำ
- ผลลัพธ์ที่คาดหวัง
- Screenshots (ถ้ามี)
